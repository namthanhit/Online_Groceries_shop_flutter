## 🔧 Cài đặt và chạy ứng dụng

### 1. Clone repository
```bash
git clone https://github.com/namthanhit/Online_Groceries_shop_flutter
cd Online_Groceries_shop_flutter

```

### 2. Cài đặt dependencies
```bash
flutter pub get
```

### 3. Chạy ứng dụng
```bash
flutter run
```
